Audiencia

Para quién es y para quién no.

Es ideal para

Clínicas, laboratorios, consultorios médicos, centros de especialidades y pequeñas o medianas empresas que tratan datos personales y necesitan iniciar su cumplimiento con la LOPDP de forma ordenada, práctica y profesional. Ideal para organizaciones que todavía no cuentan con políticas, avisos de privacidad, procesos internos, contratos actualizados ni controles básicos para proteger datos de pacientes, clientes, colaboradores o proveedores.

No es para

Empresas que buscan únicamente descargar plantillas sin acompañamiento profesional. Tampoco para organizaciones que requieren una implementación avanzada, acompañamiento permanente como DPO externo, auditorías técnicas profundas, certificaciones ISO, defensa en procesos sancionatorios o gestión integral de ciberseguridad.

Entregables

Qué incluye.

Diez entregables concretos, no plantillas genéricas. Cada documento se construye a partir de tu realidad, no de un copy-paste.

✓

Registro de Actividades de Tratamiento (RAT)

Inventario detallado de todos los tratamientos de datos que realiza tu organización.

✓

Aviso de Privacidad

Para sitio web y formularios — redactado para tu operación específica.

✓

Regularización de consentimientos

Formatos y evidencias para recolección y registro válido de consentimientos.

✓

Contratos y cláusulas LOPDP

Para vínculos con encargados, proveedores y aliados externos.

✓

Medidas técnicas urgentes

Recomendaciones sobre accesos, contraseñas y controles básicos de seguridad.

✓

Procedimiento ARCO+

Atención a derechos de los titulares: acceso, rectificación, oposición, cancelación y otros.

✓

Política interna de protección de datos

Marco normativo interno alineado con la LOPDP y la realidad de tu empresa.

✓

Capacitación inicial obligatoria

Una sesión para todo el equipo, con materiales de referencia.

✓

Matriz de riesgos de seguridad (base)

Identificación y priorización de los riesgos legales, técnicos y organizacionales detectados.

✓

Auditoría interna + Informe final

Revisión de avance e informe gerencial con hallazgos, acciones y recomendaciones.

Transparencia

Qué no incluye.

Preferimos decirlo antes que después. Si necesitas algo de esta lista, lo hablamos como servicio adicional o te orientamos al plan correcto.

  • Servicio permanente como Delegado de Protección de Datos externo
  • Representación legal ante la Superintendencia de Protección de Datos
  • Evaluación de Impacto completa (salvo contratación adicional)
  • Revisión ilimitada de contratos o documentos fuera del alcance inicial
  • Acompañamiento mensual posterior a la entrega final
  • Implementación integral avanzada para empresas con múltiples sedes o estructuras complejas
